Another Ritual

Bienvenue sur Another Ritual, un serveur d'émulation pour In Memoriam le dernier rituel.
Le serveur est actuellement en développement et permet simplement de lancer le jeu en contournant la création de compte par le web.

Attention : Vous devez posséder le jeu original, ce service ne fourni pas de crack ni de patch no-cd. Vous devez impérativement utiliser votre copie personnelle du jeu.

Le projet est 100% open-source et respecte votre vie privée, vous pouvez utiliser notre serveur ou l'héberger vous-même !

Utiliser notre serveur

Pour utiliser notre serveur, vous devez modifier le fichier hosts de votre système. Pour se faire, veuillez consulter cet article. Vous devez insérer les entrées suivantes : 

51.15.129.72 inmemoriamdev.com 
51.15.129.72 www.inmemoriamdev.com

Une fois les entrées rajoutées, essayer de vous rendre sur http://www.inmemoriamdev.com
Vous devriez voir une page avec le nom du serveur, si ce n'est pas le cas, veuillez vérifier votre fichier host

Ce qui fonctionne

  • Création de compte
  • Accès au DVD du Phoenix

Ce qui ne fonctionne pas

  • Envoi de mails
  • Envoi de SMS

Aide

Lancer le jeu, lors de la création du compte, répondez ce que vous voulez aux premieres questions. Renseignez ensuite une adresse mail et un utilisateur.
Cliquez ensuite sur le bouton retour au lieu de "ouvrir ma messagerie", dans le menu principal, cliquez sur "Confirmer mon enregistrement au ca où il ne serait pas validé" et taper un mot de passe.
Cliquer sur valider et vous avez accès au DVD du Phoenix
